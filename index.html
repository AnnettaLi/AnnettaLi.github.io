模拟shift，重构myShift代码

出队，将数组中第一个元素取出来
参数：none
返回值：出队的元素
原数组第一项被删除

代码：

Array.prototype.myShift = function () {
// 拿到数组第一个数
var num = this[0];
// 将数组的第二项赋值给第一项，第三项赋值给第二项…
// this[0] = this[1];
// this[1] = this[2];
// this[2] = this[3];
for(var i = 0;i<this.length;i++){
this[i] = this[i+1];
}
// 通过length属性删除最后一项
this.length = this.length-1;
return num;
};
var arr = [1,2,3,4];
var result = arr.myShift();
console.log(arr);
console.log(result);

关于Javascript(2)

五. 操作符

1.算术操作符：+、-、*、/、%， +=、-=、 *=、/=、%=

2.赋值操作符

  • result += 3 ； 等价于 result = result + 3;
  • var result = 1 + 2; 将1+2这个表达式的运算结果赋值给result

3.一元运算符

​ ① ++ 自增 – 自减

  • 前置：先自增、减再参与其他运算

  • 后置：先参先其他运算，再自增、自减

​ ②+ - （将其他数据类型转换为数字类型）

4.逻辑运算符

  • &&逻辑与 （短路运算符）：【返回结果不一定是boolean】一假则假全真为真

  • ||逻辑或 ：一真则真全假为假

  • ! ：将其他数据类型转换为boolean

    5.比较运算符（比较栈区的值）【返回类型为Boolean】

  • ==、!=: 比较值

  • ===、!==: 先比较数据类型，如果数据类型不一致，会直接返回false，如果数据类型一致，再比较值。

  • <、<=、>、>=

6.三目运算符

​ exp1 ? exp2 : exp3 （当exp1为真，返回exp2，否则返回exp3）

7.位运算符

​ 异或 ^ 、位与& 、位或 |

▲&&与&的区别：&&是逻辑运算符 / &是位与运算符。

8. 拼接运算符

​ 当操作数有字符串的时候表示拼接 。当使用 “+”，操作数中就出现了字符串。

六.类型转换

1.其他数据类型转换为number

方式一：使用Number()函数

▲▲▲注意：如果是字符串是纯数字，则转换为数字；如果字符串中有非数字，则转换结果为NaN；如果字符串为空串或者是空格，则结果为0；如果是Boolean类型，true为1，false为0；如果是null,结果为0；如果是undefined，结果为NaN.

方式二：+v、-(-v)

方式三: parseInt()方法 ：把一个字符串转换为整数。可以将一个字符串中的整数内容解析出来，如parseInt(“123.123a123”) 结果为123。

​ parseFloat()，把一个字符串转换为浮点数。可以获得有效的小数。

原文链接：https://blog.csdn.net/qq_33334214/article/details/80810505

2.其他数据类型转换为boolean

方式一：使用Boolean()函数 【null false，”” false，NaN false，undefined false】

方式二：!!

3.其他数据类型转换为string

方式一：使用string()函数

方式二：v+””

方式三：v.toString(); //null与undefined除外

七.流程控制语句

1. 分支语句：if else elseif，switch

2.循环语句：基本for循环语句，前置判断循环 while循环，后置判断循环 do-while循环
