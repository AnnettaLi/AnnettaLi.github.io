关于Javascript(1)

一. Javascript组成

Javascript由三部分组成：

1）ECMAScript5（javascript语法标准）

2）DOM （document object model）
js操作html的api

3）BOM （browser object model）
js操作浏览器的api

二. Javascript解释器

1) Javascript在所有的主流浏览器中都具备js解释器
js -> 浏览器（客户端）

2）js不仅仅可以运行在浏览器端，还可以运行在服务器端

  • js运行在浏览器端：动画，表单验证，ajax数据交互…
  • js运行在服务器端：js转换，代码编译，操作数据库，流，网络, iot

三. Javascript运用

1）创建一个文件并进入vim
vi 1-switch.js

2）编写文件

​ var day = 1；

​ switch(day){
​ case 1:
​ console.log(“周一”);
​ break;
​ case 2:
​ console.log(“周二”);
​ break;
​ …
​ default:
​ console.log(“周末”);

​ }

3) 执行代码
$ node 1-switch.js

  • console.log(a,b,c); 将a,b,c变量的内容打印到控制台中

四. 变量

  • 强类型 （变量的数据类型取决于变量的声明）
  • 弱类型（变量的数据类型取决于值的类）

  • 变量声明
    例：var a;

    var str;

  • 变量初始化
    例：a = true;

    str = 1;

  • 变量类型

    1）基本数据类型

    ​ 1.数字类型 number
    ​ var a = 3;
    ​ var a = 3.0;
    ​ var a = 011;
    ​ var a = 0x11;

    ​ 2.字符串类型 string
    ​ var str = “hello world”;
    ​ var str = ‘hello’;
    ​ var str = ‘{“name”:”terry”}’

    ​ 3.布尔类型 boolean
    ​ var b = true;
    ​ var b = false;

    ​ 4.null object
    ​ 一般表示空对象
    ​ var a = null;

    ​ 5.undefined
    ​ var a; // undefined
    ​ var a = undefined; //undefined

    2）引用数据类型

    ​ 1.数组

    ​ 2.对象

    ​ 3.函数

    ​ 4.正则表达式

    3）检测数据类型

    • typeof a

    • isNaN(result)

      如果result是NaN,那么这个函数返回true。 NaN :不是数值

    • isFinite(result)

      如果result是一个有穷数，返回true

      4）基本数据类型与引用数据类型在内存中的表示

    • 基本数据类型的值保存在栈区

    • 引用数据类型的引用地址保存在栈区，内容保存在堆区
