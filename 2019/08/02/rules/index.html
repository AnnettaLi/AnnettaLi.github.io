CSS规则

字体

  • font-family 为文字制定特殊的字体

  • font-size :12px; 字体大小

  • line-height：2em； 行高【文本垂直居中】

文本

  • color:#333; 字体颜色
  • text-align:center; 文本水平居中
  • text-indent：2em； 文本缩进
  • text-shadow：12px 12px 3px #ccc; 文本阴影

列表样式

​ list-style：none；

盒子样式

  • heigh

  • width

  • margin 外边距

    • margin:10px; 上右下左
    • margin:0 10px; 上下为0，左右10px
    • margin: 0 5px 10px; 上0，左右5px，下10px
    • margin:0 5px 10px 15px; 上右下左

  • padding 内边距

  • border 边框

    • border-radius：5px； 圆角半径

    • border-style：solid dotted double dashed

      ​ 实线 点状线 双线 虚线

    • 简写：border：1px solid red；

      ​ 边框为1px，红色的实线

  • background-image：url（）； 背景图片

  • background-color 背景颜色

  • 盒模型的分类：box-sizing

    • content-box 【内容盒子、标准盒子、w3c盒子】
    • border-box 【边框盒子、IE盒子、怪异盒子】

单位

颜色

  • #333333
  • #ededed
  • rgb（0,0,0）
  • rgba（0，0, 0, 0.3） 渐变色

长度

  • 绝对单位 px
  • 相对单位
    • em 相对于当前元素
    • % 相对于父元素
