函数

1.作用

(1)特定功能的封装

(2)构造函数，创建对象模板

2.函数的定义

(1)函数声明: function 函数名(形参) {
函数体
}

(2)函数表达式: var 函数名 = function(形参){};

3.调用方式

函数名(实参);
函数名.call(this,实参列表);
函数名.apply(this,实参数组);

4.函数声明会提升

如果一个函数使用函数声明的方式来定义，那么在函数定义之前就可以调用该函数。

5.函数的作用域

(1)函数内部申明的变量，函数外部无法访问

(2)函数执行完毕后，函数内部的变量会被释放

(3)函数内部不存在局部作用域

(4)函数内部的变量也会被提升

6.函数内部属性

(1)形参

(2)arguments //接受实参的真正所在，类数组对象

(3)this //当前对象（环境对象）

7.值传递和引用传递

值传递：var a = 3;var b = a;a++;

引用传递：var a = {age:12,name:”tom”};var b = a; a.age = 12;

8.函数应用

(1)值

(2)参数：回调函数

(3)返回值：定制函数
